# Atticus Project - Next.js Site Files

## Where to place each file in your my-site folder:

```
my-site/
├── app/
│   ├── layout.tsx          ← replace with: layout.tsx
│   ├── page.tsx            ← replace with: page.tsx
│   ├── globals.css         ← replace with: globals.css
│   ├── components/
│   │   ├── Navbar.tsx      ← new file: Navbar.tsx
│   │   └── Footer.tsx      ← new file: Footer.tsx
│   ├── research/
│   │   └── page.tsx        ← new file: research-page.tsx
│   └── contact/
│       └── page.tsx        ← new file: contact-page.tsx
```

## Steps:
1. Create folder: app/components/
2. Create folder: app/research/
3. Create folder: app/contact/
4. Place files as shown above
